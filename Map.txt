Map 객체는 키와 값의 쌍으로 이루어진 컨렉션이다. 객체와 유사하지만 차이가 있다. 
Map 객체는 객체를 포함한 모든 값을 키로 사용할 수 있으며 이터러블할 수있다. 
요소 개수를 확인하고자 할 때는 map.size를 사용한다. 

Map 생성자 함수는 인터러블 인수로 전달받아 map 객체를 생성한다. 이때 인수로 전달되는 이터러블은 키와 값의 쌍으로 이뤄진 요소로 구성되어야 한다. 
중복된 키를 갖는 요소가 존재하면 값이 덮어써진다. 따라서 중복된 키를 갖는 요소가 존재할 수 없다.

요소 순회 
map객체의 요소를 순회하려면     Map.prototype.forEach 메서드를 사용한다. 
콜백함수와 forEach 메서드의 콜백 함수 내부에서 this 로 사용될 객체를 인수로 전달한다.
